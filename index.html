HTML 5

HTML5 é a versão mais recente da linguagem de marcação de hipertexto (HTML), utilizada para criar páginas web.

Caracteristicas

  • É uma linguagem de marcação
  • É uma linguagem interpretada
  • Roda no lado cliente
  • Não é case-sensetive
  • É formada por tags
